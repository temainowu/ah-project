
In this folder there are several files.

install_flask.ps1 is a Powershell script that will install flask and its dependencies.

run.ps1 is a Powershell script that will run the flask server contained in server.py.

To run these scripts, you should open a powershell window inside the "ah-project" folder.

Then, you can run the following commands:
.\install_flask.ps1
.\run.ps1
